SISTEMA CRUD

O sistema CRUD é uma estrutura fundamental no desenvolvimento de software como uma base para a interação entre o usuario e o banco de dados, uma forma de simplificar o processo de construção de um aplicativo de forma que ainda assim eles possam ser gerenciados de forma completa

A letra C em CRUD significa "create", em português, criar, é o primeiro passo ao gerir dados, se refere a funcionalidade de adicionar nossas informações dentro de um sistema de armazenamento, ao por exemplo fazer login em um site você adiciona suas informações a algum sistema que gere seus dados e depois processados

A letra R em CRUD significa "read", em português, ler, ela se refere a funcionalidade que permite os usuarios recuperem e visualizem os dados criados, como uma lista de resultado de uma pesquisa ou no acesso de uma pagina web

A letra U em CRUD significa "update", em portugues, atualizar, se refere a funcionalidade de modificação de dados que ja existem, quando se troca uma senha ou e-mail, quando se altera o preço de um produto, o estado de algum objeto, todos utilizam do Update

A letra D em CRUD significa "delete", em português, deletar, é a ultima funcionalidade e se refere a capacidade de excluir dados de dentro do sistema, liberando espaços e garantido registros obsoletos ou desnecessarios no sistema

Estes são os elementos que compoem um sistema CRUD que está em todo lugar, sendo um dos componentes mais importantes na criação de qualquer sistema
